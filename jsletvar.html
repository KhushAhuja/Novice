Basics

let|var|const

Before defining a variable you might be seeing these keywords that i was using. So there are three types of these keywords and we'll see which one to use when.

const - As the name suggests it is for declaring a constant. 

            
const pi=3.14;

Now this pi cannot be changed in the whole program. And if we try to do so we'll no doubt see an error.
What if we just declare a constant but didn't defined it - 

            
const pi;

In Js this line of code will give an error too. It's scope is block scope. We'll discuss about scopes in detail in next section.



let - It is used to define a variable

            
let skill='JavaScript';
console.log(typeof skill);
skill=2;
console.log(typeof skill);

Here the 2nd command will print the datatype of 'skill' according to 1st command which will be 'String' and the 4th command will print the datatype of 'skill' after processing 3rd command which will be 'Number'. This means that we can redefine a variable which was declared using let.
