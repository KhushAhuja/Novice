History

Oh! Ofcourse JavaScript as the name implies it must have something to do with Java programming language, isn't it what you were thinking?
But the actual answer is NO. Actually what happened, initially name of this language was 'Mocha', then it was changed to 'LiveScript'. In 1995 Java was so popular. So inorder to make LiveScript popular the committee decided to change its name from LiveScript to 'JavaScript'.
Overall JavaScript has nothing to do with Java.

By Learning JavaScript you open the gates to learn many new technologies too like Nodejs, React.js, Angular, ReactNative and many more...



EcmaScript is the international standard that defines JavaScript.


From 2015 the modern JavaScript era had begin. In 2015 the committee decided that they will issue little updates in the language every year rather than issuing a big update suddenly as they did back in 2015.


JavaScript is backwards compatible, this means all websites built before 2015 when ES6 version was launched starting the modern JavaScript era can be run today in modern browsers.
older features of JavaScript were not removed from the modern JavaScript.
