Basics

Data Types

Values are of two types -


Primitive Value -

            
let mark='web developer';

Here 'web developer' is a value which is termed as primitive.
Object Value - 

            
let profession = {
    mark:'web developer'
};

Here 'web developer' is an object value. We will study about objects in later sections.



Data type, in programming, defines which type of value a variable contains. In a way it defines the type of data a variable has.
In other programming languages, data types are defined for variables, like we say that this is an integer type variable which means that particular variable is storing an integer. But in JavaScript, data types are defined for values. Here we say that this variable is containing a value of integer type.
It is a little confusing but i hope you understood what i was trying to say.



Here we will study primitive data types. We will see non-primitive data types in later sections.

String : Sequence of characters 

            
let name='Mark';

Number : Used for decimals and integers

            
let age=30;

Boolean : Logical type that can only be true or false

            
let married=true;

Null : Also means empty value

            
let kids=null;

Undefined : Value taken by a variable that is not yet deifined

            
let name;


There are two more types of primitive data types issued in modern JavaScript -

Symbol (ES 2015) : Value that is unique and cannot be changed

            
const name1=Symbol('Mark');
const name2=Symbol('Mark');

name1 and name2 are not equal.


BigInt (ES 2020) : Larger integers than the Number type can hold
