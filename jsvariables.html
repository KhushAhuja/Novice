Variables

Memory of our computer is like a locker shelf that has many cabins.


We usually fill these cabins with our stuff in real life similarly in a computer memory we can fill these cabins with values.

Now what can be these values?
These can be integers, decimals, characters etc.

These cabins have a number printed on it like cabin no.202, so we can say that 202 is the address of that cabin. Similarly in computer memory each block or cabin has a unique number associated with it which is it's address.

Now what is a Variable?
Let say there is cabin no. 202 owned by Mark Zuckerburg and he named it as Zuckerburg's Locker , so here cabin is a memory block, 202 is the address of that memory block and Zuckerburg's Locker is acting as a variable name assigned to that memory block.

So a variable is nothing but a memory block whose value can be changed anytime, obviously it is a variable that means something that can be changed. 

            
let apples=3;

Here apples is a variable name assigned to a memory block and 3 is the value in it.
